# Recharge APK Project

This source package includes:
- Expo React Native app
- Firebase Auth + Firestore setup
- Razorpay checkout integration scaffold

## Before running
Fill these files with your real keys:
- firebaseConfig.ts
- razorpayConfig.ts

## Commands
npm install
npx expo install react-native-razorpay
npx expo prebuild
npx expo run:android

## Important
This zip is source code, not a compiled APK.
To create a real APK, you need to build it locally or with EAS, and use your own Firebase and Razorpay credentials.
